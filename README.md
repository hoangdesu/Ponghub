## Installation
If you want to compile the game yourself, you'd need Lua and L√ñVE2D framework.
If you are on a Mac, you can install using Brew:
1. Install Lua:
```
brew install lua
```

2. Install L√ñVE2D:
```
brew install --cask love
```

3. Add love alias to your PATH:
```
alias love="/Applications/love.app/Contents/MacOS/love"
```

4. Clone the project:
```
git clone https://github.com/hoangdesu/Ponghub.git
```

5. Go to the project and run the game:
```
cd Ponghub/src
love .
```
